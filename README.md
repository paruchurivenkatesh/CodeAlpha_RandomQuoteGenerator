# CodeAlpha_RandomQuoteGenerator
## 🌈 Random Quote Generator

A fun and colorful Random Quote Generator web application built using HTML, CSS, and JavaScript.

This project was created as part of the CodeAlpha App Development Internship.

---

# ✨ Features

- 🎯 Generates random motivational quotes
- 🔁 New quote appears on every button click
- 🚫 Prevents immediate repeated quotes
- 🌈 Kids-friendly colorful design
- ☀️ Animated sun background
- ☁️ Moving clouds animation
- 🐦 Flying birds animation
- ⛰️ Mountain scenery
- 🌊 Animated river effect
- 📱 Responsive design for mobile and desktop

---

# 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

# ▶️ How to Run the Project

## Method 1: Using Browser

1. Download or clone the repository
2. Open the project folder
3. Double click on `index.html`

---

## Method 2: Using VS Code Live Server

1. Open project folder in VS Code
2. Install "Live Server" extension
3. Right click on `index.html`
4. Click "Open with Live Server"
